Corrige complètement les différences visuelles entre la version mobile et desktop de DrimPay.

⚠️ PROBLÈME ACTUEL :
Sur mobile :
- les couleurs et le design sont modernes,
- les boutons sont bien affichés,
- les icônes sont propres,
- l’interface paraît professionnelle.

Mais sur ordinateur :
- les couleurs changent,
- certaines icônes deviennent ternes,
- les boutons perdent leur style,
- certains backgrounds deviennent fades,
- le rendu global est moins professionnel.

Je veux EXACTEMENT le même rendu visuel sur :
- mobile
- tablette
- desktop
